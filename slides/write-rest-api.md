# HTTP basics (1.1)

# Methods

* GET: idemponent (proxy-friendly)
* POST: create new resources
* PUT: replace an existing resource entirely
* PATCH: modify an existing resource partially
* DELETE: remove an existing resource

# Status code

* 2xx: succeed
    - 200 OK
    - 201 Content Created
* 3xx: redirect
    - 301 Permanent Moved
    - 302 Temporary Moved
    - 304 Not Changed
* 4xx: client error
    - 400 Bad Request
    - 401 Authentication Required
    - 403 Forbidden
    - 404 Not Found
    - 405 Method Not Allowed
* 5xx: server error
    - 500 Internal Error
    - 502 Bad Gateway

# HTTP Headers

* Content-Type: MIME Type of the body
    - application/xml
    - application/json
